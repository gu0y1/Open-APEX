OpenAPEX 机械臂操作方法、使用说明和注意事项
Author: Chen Guoyi, guoyi@comp.nus.edu.sg, NUS Computing

硬件连接：

将机械臂底座固定在厚度大于 20 毫米、长和宽大于 30 厘米的高刚性板子上（如厚木板，需根据底座固定孔位尺寸自行钻孔），并用随机械臂附带的螺丝和螺母进行紧固。

将电源适配器或电池通过 DC 头连接至机械臂的 DC 口，机械臂供电电压范围为 24 至 28 伏，供电电流 15A 以上。
注意：机械臂供电后，所有关节将默认自动处于锁止状态并保持位置和姿势，此时切勿用强外力转动机械臂关节！

*机械手爪的舵机供电电压范围为 12 伏，需单独供电。

将 USB-TTL 模块插入电脑 USB 接口并正确安装驱动，打开电脑的设备管理器，查看并记录其被分配的端口号（如"COM4"）。

软件环境部署：

在Windows 系统下，解压 WPy64-31241.zip 压缩文件到本地纯英文路径的目录，得到 WPy64-31241 文件夹（注意，因为 WPy64-31241 文件夹有隐藏属性，所以解压完成后此文件夹可能不显示，如不显示，需手动更改系统设置，显示隐藏项目即可），此文件夹为控制程序运行所必需的软件环境，已经集成了完整的 Python 3.12 并且在内部编译、安装和配置好了软件运行所需的所有功能包和依赖库，解压即用。
进入文件夹后可找到 WinPython Terminal.exe 图标，右键点击后在弹出菜单中选择"以管理员身份运行..."开启 Windows PowerShell 。

解压 OpenAPEX.zip 到本地另一纯英文路径的目录， 得到 OpenAPEX 文件夹，此文件夹内容为程序的资源、配置、其他相关库和 Python 脚本文件，请勿随意增删，文件夹中的 main.py 为图形化控制程序的启动脚本文件。

在上述已经开启的 Windows PowerShell 中，通过 cd 命令更改当前目录，定位到 OpenAPEX ，使用 Python 解释器解释运行 main.py，运行命令为 python main.py 即可开启图形化界面控制程序。

机械臂控制程序启动后，机械臂串口会自动尝试建立串口连接，注意此时必须先对所有关节的磁编码器进行手动调零操作，操作方法：通过软件界面左上方的菜单进行操作，实现实物机械臂关节的转动微调，调整至机械臂实物与控制程序视图中的虚拟模型机械臂姿态完全吻合即可完成此操作，注意，此针对电机的调零操作在机械臂每次上电开启时必须重新执行一次！

机械臂操作方法：

按键盘的 上、下、左、右方向键，控制机械臂末端执行器的运动；
按键盘的 Page Up、Page Down 键，控制机械臂末端执行器的升降运动；
按键盘的 Home 键，控制机械臂回到初始位置；
按键盘的 j、l、i、k、逗号、句号键，控制机械臂末端执行器的旋转方向。

拖拽末端控件的红、绿、蓝三个箭头，可以设置末端目标位置，即半透明球体中心位置的 x、y、z 坐标。
点击中心的半透明球体，可以实时预览机械臂末端执行器的运动轨迹，并让机械臂运动到所设目标位置。
注意：控制末端的运动时，尽量小幅度进行操作，且不要超出机械臂的运动范围，否则可能造成自碰撞或碰撞外物而损坏机械臂！

保养维护：

注意：使用机械臂前后请检查线路连接，如有松动请重新固定接线端子。
